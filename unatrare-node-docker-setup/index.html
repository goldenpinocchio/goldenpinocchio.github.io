UNAT Node on Docker

by Pinocchio

Simple copy/paste setup for running a UNATRARE Node on a Windows PC with Docker Desktop.

x.com/TrustyPinocchio

As simple as - Start, Stop, Update (once Docker is set up)

Start

Use after rebooting or reopening Docker.


PowerShell
cd "$env:USERPROFILE\unatrare-node"; docker compose up -d

Stop

Pauses heartbeats, keeps saved volumes.


PowerShell
cd "$env:USERPROFILE\unatrare-node"; docker compose down

Update

Pull latest code, rebuild, restart.


PowerShell
cd "$env:USERPROFILE\unatrare-node"; git pull; docker compose build --no-cache; docker compose up -d

6 Step - Setup for Docker + UNAT Node

01

Install Docker Desktop

Install Docker, accept WSL2/Hyper-V defaults, restart if asked, then open Docker until the engine is running.

Official download
https://www.docker.com/get-started/
02

Install Git if needed

If PowerShell says git is not recognized, install Git for Windows.

Official download
https://git-scm.com/download/win
03

Clone the node

Open PowerShell or Windows Terminal and paste this.

PowerShell install line
git clone --branch unatrare/phase-0 https://github.com/joeatang/unatrare-intercom "$env:USERPROFILE\unatrare-node"; cd "$env:USERPROFILE\unatrare-node"; copy .env.example .env
04

Add public addresses

BTC is required. XCP and TAP are optional.

Open config
notepad .env

.env example
BTC_ADDRESS=your_bitcoin_address
XCP_ADDRESS=your_counterparty_address
TAP_ADDRESS=your_tap_address
05

Start the node

Builds and starts the Docker container in the background.

Start command
docker compose up -d
06

Check logs

Look for [unatrare] Node registered....

Logs
docker compose logs -f

Safety

Rule #1

Never paste a seed phrase, private key, wallet file, signing approval, or wallet password into this node setup.

The node should only need public wallet addresses for identity/reward tracking.

Docker isolates it

The node runs in a Linux container through Docker Desktop instead of directly on Windows.

Your IP may show

This is P2P software. Your IP can be visible to peers. Use a VPN if that matters.

Keep the PC awake

Heartbeats stop when Docker stops, the PC sleeps, or internet drops.
