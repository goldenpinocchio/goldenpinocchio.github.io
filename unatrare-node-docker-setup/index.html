🪟

What Docker does

Docker runs a tiny Linux environment inside Windows so the Pear-based UNATRARE node can run cleanly.

💾

What gets saved

Docker volumes keep the node identity and archive cache alive through restarts and normal updates.

🌐

What to expect

Your PC needs to stay awake. Your IP may be visible to peers, like other P2P software.

Quickstart

Six steps, no Docker lore required.

01

Install Docker Desktop

Use the official Docker download. Accept the WSL2/Hyper-V defaults, restart if prompted, then open Docker Desktop until the engine is running.

Official download
https://www.docker.com/get-started/
02

Install Git if needed

If PowerShell says git is not recognized, install Git for Windows from the official source.

Official download
https://git-scm.com/download/win
03

Clone the official node

Open PowerShell or Windows Terminal and paste this install line.

PowerShell install line
git clone --branch unatrare/phase-0 https://github.com/joeatang/unatrare-intercom "$env:USERPROFILE\unatrare-node"; cd "$env:USERPROFILE\unatrare-node"; copy .env.example .env
04

Add your public addresses

BTC address is required. XCP and TAP are optional, but useful for display and future reward logic.

Open the config
notepad .env

.env example
BTC_ADDRESS=your_bitcoin_address
XCP_ADDRESS=your_counterparty_address
TAP_ADDRESS=your_tap_address
05

Start the node

This builds the container and starts the node in the background.

Start command
docker compose up -d
06

Watch for registration

Look for [unatrare] Node registered.... Once you see that, the frog is awake.

Logs
docker compose logs -f

Daily commands

Stop, restart, or update later.

Stop the node

Pauses heartbeats, keeps saved volumes.

PowerShell
cd "$env:USERPROFILE\unatrare-node"; docker compose down

Start again

Use after rebooting or reopening Docker.

PowerShell
cd "$env:USERPROFILE\unatrare-node"; docker compose up -d

Update

Pull latest code, rebuild, restart.

PowerShell
cd "$env:USERPROFILE\unatrare-node"; git pull; docker compose build --no-cache; docker compose up -d
⚠️

Do not feed the frog your seed phrase.

The node uses public wallet addresses as identity/reward markers. It should never need a private key, recovery phrase, wallet file, signing approval, or password.
