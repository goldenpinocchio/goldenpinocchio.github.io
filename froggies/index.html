Froggies

Froggies are a 4,032-piece collection generated from the only Bitcoin difficulty periods where compact nBits contains fc. Each selected block contributes one block hash; the hash rolls the frog's look.

4,032 total Froggies 2 × 2,016-block difficulty periods fc in compact difficulty bits block hash → trait rolls

FC difficulty windows

Bitcoin retargets difficulty every 2,016 blocks. Froggies use the two historical periods whose compact difficulty field includes fc.

Blocks 266112–268127190afc85
Blocks 933408–9354231701fca1

Deterministic rolls

Leading zeroes are removed from each block hash, then the remaining hash is split into slices. Each slice maps to one visual category.

Rarity from rolls

Each slice is converted into a roll with value % 4032. Low rolls unlock the rarer looks; higher rolls map to the more common pond traits.
