FC block hash traits

Froggies are generated from the two Bitcoin difficulty windows where compact nBits contains fc. Each block becomes one Froggie. The block hash is sliced into six rolls, and the ranges below turn those rolls into traits.

Hash slicing

  • Remove leading zeroes from the block hash.
  • Split the remaining hash into six slices.
  • Map slices to body, eyes, mouth, headgear, accessory, and background.

Roll range

value % 4032 returns rolls from 0 to 4031. Lower rolls produce rarer traits.
